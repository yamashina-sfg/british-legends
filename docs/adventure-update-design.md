# Adventure Update 設計書

## 体験目標

プレイヤーはワールドマップで Bibliotheca Lodge を出発し、道沿いの作品世界へ向かう。宿屋、店、仲間、NPC会話を経由して準備を整え、洞窟・城・魔女の森など世界固有の危険地帯へ踏み込む。

## 追加ループ

Lodge（回復 / セーブ / 仲間確認 / 進化 / 店）
→ 道のあるワールドマップ
→ 作品固有の入口
→ ダンジョン探索・戦闘
→ Gold / 素材 / 経験値
→ Lodgeで装備・道具・進化

## システム

### Bibliotheca Lodge

- ワールドマップの開始地点。
- 無料セーブ、宿屋、仲間メニュー、進化、ストアを持つ。
- 司書、旅人、商人は短い会話で目的、危険、買い物の役割を伝える。

### ストア

- 武器: 鉄の剣 / 鋼の剣 / 英雄の剣
- 防具: 革鎧 / 鉄鎧 / 英雄鎧
- アクセサリー: 旅人の護符
- 道具: 回復薬 / 上級回復薬 / 蘇生薬
- Goldは敵撃破で得る。武器、防具、アクセサリーは即装備。回復薬は戦闘中に使う。

### ワールドマップ

- Lodgeから道が分岐し、Beowulfの洞窟、Hamletの城、Macbethの雷雨の森へ繋がる。
- 未解放世界は道の先を暗くし、封印表示する。

### テーマ

- Beowulf: 苔むした森、青い洞窟、古い石碑。
- Hamlet: 夜空、城壁、月光、亡霊の光。
- Macbeth: 雷雲、赤い沼、枯れ森、魔女の火。

## MVP外

- 蘇生薬の戦闘内ターゲット選択。
- 装備の売却、ランダムエンチャント、複数NPCクエスト。
