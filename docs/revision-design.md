# British Legends 修正設計書

## 目的

「ドラクエ風に見える」状態から、町とダンジョンを往復しながら判断と成長を楽しむRPGへ移行する。文学要素は問題集ではなく、敵・場所・短い演出・スキルに埋め込む。

## MVPの完成ループ

町（宿屋・武具屋・目的確認） → ダンジョン（分岐探索・宝箱・休息碑・記憶の断片） → 戦闘（防御・弱体・MP・敵の役割） → EXP / 素材 / Gold → 装備・進化 → ボス固有フェーズ → 仲間加入・次世界解放

## 実装方針

### 町

- 各作品世界の入口に小さな町を置く。
- 宿屋はGoldを消費してHP/MPを全回復する。
- 武具屋は世界ごとの剣・鎧を販売する。
- 装備はデータ化し、戦闘ステータスに加算する。

### 戦闘

- 通常攻撃、特技、防御に加え、敵の防御低下とボスの怒りフェーズを使用する。
- DragonはHP半分で怒り、攻撃力が上がり、火炎ブレスの使用頻度が増える。

### 探索

- 自動生成洞窟の枝道に宝箱、休息碑、記憶の断片を配置する。
- 休息碑は一時回復、記憶の断片は短い文学的な一文を表示する。

### 成長

- Gold、装備、素材、進化を別の価値として扱う。
- 戦闘結果にはEXP、Gold、素材、レベルアップをまとめて表示する。

## データ追加

- `Equipment`: price, slot, stat bonus, worldId
- `OwnedCharacter`: equippedWeaponId, equippedArmorId
- `SaveData`: gold
- `MapEntity`: rest, memory
